
# Quick Start Guide

## Overview
1. Create virtual switch using `create-v-switch.ps1`
2. Create VM using `create-pihole-vm.ps1`
3. Install Ubuntu and Pi-hole
4. Configure the router
5. Access Pi-hole admin at `http://192.168.1.10/admin`

## Expected Outcome
- Ads blocked on all network devices
- Web interface accessible for management
- DNS queries filtered through Pi-hole
